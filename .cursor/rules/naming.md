# Naming Conventions

Folders:
lowercase

Files:
camelCase

Classes:
PascalCase

Interfaces:
IUser

Enums:
UserStatus

Models:
User

Repositories:
UserRepository

Services:
UserService

Controllers:
UserController

Variables:
camelCase

Functions:
camelCase

Constants:
UPPER_SNAKE_CASE

MongoDB schema fields:
camelCase

Collection names:
lowercase plural (e.g. users)
